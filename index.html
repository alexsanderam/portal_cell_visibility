Portal-cell visibility

Instruções de uso

  1. Clique com o mouse sobre o SVG para criar os sites que representarão as células.
  2. Pressione a tecla espaço para computar o diagrama de Voronoi e, então, de fato definir as células
Após estes passos, pode-se definir a posição do observador ou criar/remover portais:
  • Para definir a posição do observador dê um duplo clique em qualquer célula.
  • Para criar um portal, posicione o cursor do mouse sobre uma aresta do diagrama de Vornoi e dê um clique.
  • Para remover um portal, posicione o cursor do mouse sobre o portal o qual deseja-se remover, e dê um duplo clique.
  • Para aumentar ou diminuir o tamanho de um novo portal pressione as teclas '+' ou '-', respectivamente.
